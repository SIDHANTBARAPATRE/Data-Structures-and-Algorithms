# Dynamic Programming (DP) in C++

This repository contains my solutions to **Dynamic Programming (DP)** problems implemented in **C++**. It is part of my **Data Structures and Algorithms (DSA)** learning journey and focuses on understanding problem-solving techniques, optimization, and coding interview preparation.

## 📚 Topics Covered

* Fibonacci Sequence
* Climbing Stairs
* Frog Jump
* House Robber
* 0/1 Knapsack
* Unbounded Knapsack
* Longest Common Subsequence (LCS)
* Longest Increasing Subsequence (LIS)
* Matrix Chain Multiplication
* Partition DP
* DP on Grids
* DP on Trees
* DP on Strings
* Digit DP
* Bitmask DP
* Stock Buy & Sell Problems
* Other Interview and Competitive Programming Problems

## 🚀 Dynamic Programming Approaches

For each problem, I try to implement one or more of the following approaches:

1. Recursive Solution
2. Memoization (Top-Down DP)
3. Tabulation (Bottom-Up DP)
4. Space Optimized DP (where applicable)
